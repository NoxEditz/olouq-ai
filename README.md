## 📖 نظرة عامة

**علوق الثانوية** هي منصة تعليمية متقدمة مدعومة بالذكاء الاصطناعي، مصممة خصيصاً لطلاب الثانوية العامة المصرية. توفر المنصة مساعداً ذكياً شاملاً مع أدوات دراسية متكاملة لتحقيق التفوق الأكاديمي.

## ✨ المميزات الرئيسية

### 🤖 مساعد ذكاء اصطناعي متقدم
- **نظام ذكي متعدد النماذج**: يستخدم 13+ نموذج AI مختلف للحصول على أفضل النتائج
- **نظام احتياطي تلقائي**: عند انتهاء حصة نموذج، يتحول تلقائياً للنموذج التالي (لا توقف أبداً!)
- **شرح بالعامية المصرية**: يفهم ويرد بالعامية المصرية بشكل طبيعي
- **دعم جميع المواد**: رياضيات، فيزياء، كيمياء، أحياء، لغات، وأكثر
- **تحليل الصور**: رفع صور للمسائل وحلها مباشرة
- **شرح تفصيلي**: شرح خطوة بخطوة مع أمثلة عملية

### 🛠️ الأدوات الدراسية المتقدمة

#### 📚 أدوات التعلم
- ✅ **اختبارات تفاعلية**: اختبارات ذكية مُخصصة حسب المادة والموضوع
- 🗂️ **بطاقات تعليمية (Flashcards)**: نظام بطاقات ذكي للحفظ والمراجعة
- 📖 **دفتر الملاحظات**: حفظ وتنظيم الملاحظات المهمة
- 🎯 **وضع الامتحان**: تدريب مكثف وتجهيز للامتحانات

#### 🧮 أدوات حسابية
- 🧮 **آلة حاسبة علمية**: آلة حاسبة متقدمة لجميع العمليات الحسابية
- ∑ **حل المعادلات**: حل المعادلات الرياضية والفيزيائية تلقائياً
- 📐 **حسابات هندسية**: أدوات لحساب المساحات والحجوم

#### ⏱️ إدارة الوقت
- ⏱️ **مؤقت بومودورو**: نظام بومودورو للدراسة المركزة (25 د مذاكرة / 5 د راحة)
- 📅 **جدول المذاكرة**: جدولة وتنظيم أوقات الدراسة
- ⏰ **تذكيرات ذكية**: تذكيرات تلقائية للمراجعة

#### 📊 متابعة التقدم
- 📈 **متابعة التقدم الدراسي**: رسوم بيانية لتتبع الأداء
- 🏆 **نظام الإنجازات**: نقاط وشارات تحفيزية
- 🔥 **سلسلة الأيام**: تتبع الأيام المتتالية للمذاكرة
- 📊 **إحصائيات تفصيلية**: ساعات الدراسة، المواضيع المُغطاة، إلخ

#### 🔬 مراجع علمية
- 🧪 **الجدول الدوري التفاعلي**: جدول دوري شامل مع معلومات تفصيلية
- 📐 **ورقة القوانين**: قاعدة بيانات لجميع القوانين والمعادلات
- 📚 **مكتبة الأمثلة**: أمثلة محلولة لجميع المواد

### 🎙️ مميزات صوتية
- 🎤 **إدخال صوتي**: اسأل بصوتك بدلاً من الكتابة
- 🔊 **قراءة الإجابات**: الاستماع للإجابات بدلاً من القراءة
- 🔴 **وضع لايف**: محادثة صوتية تفاعلية مع المساعد الذكي
- 🎯 **كشف الصوت التلقائي**: يبدأ الاستماع تلقائياً عند توقفك عن الكلام

### 💬 نظام المحادثات
- ✅ **محادثات متعددة**: إنشاء محادثات منفصلة لكل موضوع
- 📂 **تصنيف حسب المواد**: تنظيم المحادثات حسب المادة الدراسية
- 💾 **حفظ تلقائي**: جميع المحادثات محفوظة محلياً
- 📥 **تصدير المحادثات**: حفظ المحادثات كملفات نصية

### 🎨 تجربة مستخدم متميزة
- 🌙 **تصميم داكن أنيق**: واجهة عصرية بألوان نيون خضراء

## 🔧 التثبيت والإعداد

### المتطلبات
- حساب Vercel (مجاني)
- واحد أو أكثر من مفاتيح API التالية:
  - Google Gemini API Key (موصى به)
  - Groq API Key (موصى به)
  - OpenRouter API Key (اختياري)
  - Together AI API Key (اختياري)
  - Hugging Face API Key (مجاني - احتياطي)

### خطوات التثبيت

#### 1. استنساخ المشروع
```bash
git clone <your-repo-url>
cd enhanced-project
```

#### 2. الحصول على مفاتيح API

**Google Gemini API** (موصى به - مجاني):
1. اذهب إلى: https://makersuite.google.com/app/apikey
2. انقر "Create API Key"
3. انسخ المفتاح

**Groq API** (موصى به - مجاني وسريع جداً):
1. اذهب إلى: https://console.groq.com/keys
2. سجل حساب جديد
3. انقر "Create API Key"
4. انسخ المفتاح

**Hugging Face** (مجاني - احتياطي عالمي):
1. اذهب إلى: https://huggingface.co/settings/tokens
2. انقر "Create new token"
3. اختر نوع "Read" وانسخ المفتاح

**OpenRouter** (اختياري - للنماذج الإضافية):
1. اذهب إلى: https://openrouter.ai/keys
2. سجل حساب جديد
3. انقر "Create Key"
4. انسخ المفتاح

**Together AI** (اختياري):
1. اذهب إلى: https://api.together.xyz/settings/api-keys
2. سجل حساب جديد
3. انشئ API Key
4. انسخ المفتاح

#### 3. الرفع على Vercel

**الطريقة الأولى - من خلال GitHub:**
1. ارفع المشروع على GitHub
2. اذهب إلى https://vercel.com
3. انقر "Import Project"
4. اختر المستودع الخاص بك
5. أضف متغيرات البيئة (Environment Variables):
   - `GEMINI_API_KEY`: مفتاح Google Gemini
   - `GROQ_API_KEY`: مفتاح Groq
   - `OPENROUTER_API_KEY`: مفتاح OpenRouter (اختياري)
   - `TOGETHER_API_KEY`: مفتاح Together AI (اختياري)
6. انقر "Deploy"

**الطريقة الثانية - من خلال Vercel CLI:**
```bash
npm install -g vercel
vercel login
vercel

# أضف المتغيرات في لوحة التحكم أو:
vercel env add GEMINI_API_KEY
vercel env add GROQ_API_KEY
# ... إلخ
```

#### 4. إعداد النطاق المخصص (اختياري)
1. في لوحة تحكم Vercel، اذهب إلى Settings > Domains
2. أضف نطاقك المخصص
3. اتبع التعليمات لتوجيه DNS

## 🔐 نظام النماذج المتعددة

### كيف يعمل؟
المنصة تستخدم **نظام احتياطي ذكي** مع 11+ نموذج AI:

1. **الأولوية الأولى**: Gemini 2.0 Flash (الأحدث والأسرع)
2. **الأولوية الثانية**: Gemini 1.5 Flash
3. **الأولوية الثالثة**: Gemini 1.5 Flash 8B
4. **الأولوية الرابعة**: Llama 3.3 70B (Groq)
5. **الأولوية الخامسة**: Llama 3.1 70B (Groq)
6. **الأولوية السادسة**: Mixtral 8x7B (Groq)
7. **نماذج احتياطية**: OpenRouter (GPT-4, Claude, Gemma)
8. **نماذج إضافية**: Together AI (Llama 405B, Qwen)

### مميزات النظام
✅ **لا توقف أبداً**: إذا انتهت حصة نموذج، ينتقل تلقائياً للتالي  
✅ **نظام تبريد ذكي**: النماذج المنتهية تُعطى فترة راحة 5 دقائق  
✅ **اختيار تلقائي**: يختار دائماً النموذج الأفضل المتاح  
✅ **تسجيل مفصل**: يمكنك رؤية أي نموذج تم استخدامه  

### إضافة المزيد من النماذج
ببساطة أضف المزيد من مفاتيح API في متغيرات البيئة، والنظام سيستخدمها تلقائياً!

- ✅ **تخزين محلي**: جميع بياناتك محفوظة على جهازك فقط
- ✅ **لا قاعدة بيانات**: لا يتم حفظ أي بيانات على الخادم

- ✅ **تنظيف المدخلات**: حماية من XSS attacks
- ✅ **مفاتيح آمنة**: جميع مفاتيح API محمية في متغيرات البيئة

## 🐛 إصلاح المشاكل

### المشكلة: "لا يمكن التواصل مع الذكاء الاصطناعي"
**الحل**: 
- تأكد من إضافة واحد على الأقل من مفاتيح API
- تحقق من صحة المفاتيح
- انتظر دقيقة وحاول مرة أخرى (قد تكون حد الطلبات)

### المشكلة: "تم تجاوز الحد المسموح"
**الحل**:
- الحد الحالي: 30 رسالة/دقيقة
- انتظر دقيقة واحدة
- أو أضف المزيد من مفاتيح API لزيادة السعة

### المشكلة: الصوت لا يعمل
**الحل**:
- تأكد من السماح للموقع باستخدام الميكروفون
- جرب في متصفح آخر (Chrome موصى به)
- تحقق من إعدادات الخصوصية

### المشكلة: رفع الصور لا يعمل
**الحل**:
- تأكد من أن لديك مفتاح Gemini API (الوحيد الذي يدعم الصور)
- تأكد من أن الصورة بصيغة: JPG, PNG, WEBP
- حجم الصورة أقل من 5MB
