## 🚀 كيفية الاستخدام

### الخطوة 1: الحصول على مفاتيح API

**الحد الأدنى (مطلوب واحد فقط):**
- Google Gemini: https://makersuite.google.com/app/apikey
- أو Groq: https://console.groq.com/keys

**للأداء الأمثل (موصى به):**
- أضف كل المفاتيح المتاحة لك
- كلما زادت المفاتيح، زاد الاستقرار

### الخطوة 2: النشر على Vercel

**طريقة سريعة:**
1. ارفع المشروع على GitHub
2. اذهب لـ vercel.com
3. Import المشروع
4. أضف المتغيرات:
   ```
   GEMINI_API_KEY=your_key
   GROQ_API_KEY=your_key
   (اختياري) OPENROUTER_API_KEY=your_key
   (اختياري) TOGETHER_API_KEY=your_key
   ```
5. انقر Deploy

**التفاصيل الكاملة:** اقرأ `DEPLOYMENT.md`
